# sales_data_analysis
This project is to analyze the sales dataset in Google Sheets, through which it will uncover sales trends, order performance, and revenue insights. The main purpose of the project is to build an interactive dashboard showing visualizations of key metrics necessary for data-driven decisions.

## Tools Used
- Google Sheets
- Excel / CSV
- Data Visualization

  ## Key Insights
- Monthly sales trends
- Top-performing products
- % contrbution by city
- product by sales
